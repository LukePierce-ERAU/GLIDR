Open-source software for design single skin paragliders and kites.

Home page
http://sourceforge.net/projects/singleskin


Capabilities:

  • All-in-one software to design single skin wings
  • Simple editor of wing shape
  • Different triangles types, bridles and printing modes
  • Allow print small A4 papers and glue them together to get scale 1:1
  • 3d model and print templates can be opened as DXF files in any CAD like Google SketchUP Pro, Autocad, Corel Draw, etc


How to desing simple wing:

  • First, define Area, AR and Number of Cells.
  • Second, enter approximate Glide Ratio of your future wing. Do not type any value you want, be realistic. See wings with the same area, AR and num of cells and type close Glide Ratio. Usually kites and speedriders with low AR have Glide Ratio 4, speedflyers 5-6 and paragliders about 7-7.5 (single skin wings have 1-1.5 lower GR than double skin).
  • Now go to page "Wing shape" and draw shape as you wish. Select 3d view tab and see what happens. This viewer cannot show polygonal model, so you can open file "wing_3d.dxf" (in the program folder) in any CAD like Google SketchUP Pro.
  • Return back to main page and choose different options, number of lines at cascades and so on.
  • When your wing is ready, go to the page Print and choose printing mode (usually separate files or overlay mode). After click Print... button, select Scale factor 1:1.
  • At last, look file "bridle.txt" to line lengths and other info.


Hints:

  • If you see popup error messages with strange text like "Windows Scripting Host", this means you entered invalid input data few second ago. Correct values and model will be recalculated automatically.
  • Hold mouse over controls, will popup expanded hint.
  • If calculation takes a long time, reduce number of cells, tune your wing and then return cells back.
  • Be careful! Each program launch rewrites DXF files and file "bridle.txt", so if you need them, copy to another place.



This software applicable for design single skin RC models, man-size paragliders and kites.


Inner 3d viewer.


Graphic vector editor of wing shape.


Curved triangles, 4 or 8 anchor points per rib, straight triangles.


Joined triangles mode (reduce lines length, but may increase drag).


Adjustable triangles height distribution in nervure.


Bridles with different num of cascades and different num lines in every level.


Separate wingtip or as main bridle.


Print templates overlay mode, all in one file or each template in separate dxf file.


Adjustable num of sewing points.

And more.


Limits:

  • Only semi 3-riser (A,B,C+D) bridle availible.
